# IBM Spyre Operator

<!-- build status will be added here -->

Spyre Operator is designed to help users and cluster administrators to configure IBM AIU Spyre cards on OpenShift cluster.
It works as a centralized one-stop operator to coordinate multiple essential components, which can hide the complexity of the Spyre enablement processes in the cluster.
